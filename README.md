# library-app
A library application for book loaning management

### Prerequisites

This application uses MongoDB as a backend [MongoDB](https://www.mongodb.com/)

## Project setup
```
npm install
```

### Run the app!
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
